prefer_const_literals_to_create_immutables

Group: style

Maturity: stable

View the Project on GitHub dart-lang/linter

  • See the Style Guide
  • List of Lint Rules

PREFER using const for instantiating list and map literal used as parameters in immutable class instantiations.

BAD:

@immutable
class A {
  A(this.v);
  final v;
}

A a1 = new A([1]);
A a2 = new A({});

GOOD:

A a1 = new A(const [1]);
A a2 = new A(const {});
