Unnecessary Brace In String Interp

Group: Style Guide

Kind: Avoid

Maturity: EXPERIMENTAL

View the Project on GitHub dart-lang/dart_lint

  • See the Style Guide
  • View On GitHub

AVOID bracketed interpolation of simple identifiers.

GOOD:

print("Hi, $name!");

BAD:

print("Hi, ${name}!");
