Declare elements in list literals inline, rather than using add and addAll methods where possible.

BAD:

var l = ['a']..add('b')..add('c');
var l2 = ['a']..addAll(['b', 'c'])

GOOD:

var l = ['a', 'b', 'c'];
var 2 = ['a', 'b', 'c'];
