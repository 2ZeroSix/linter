unnecessary_null_in_if_null_operators

Group: style

Maturity: stable

View the Project on GitHub dart-lang/linter

  • See the Style Guide
  • List of Lint Rules

Avoid null as operand in if null operator. a ?? null and null ?? a can both be replaced by a.

GOOD:

var x = a ?? 1;

BAD:

var x = a ?? null;
var y = null ?? 1;
