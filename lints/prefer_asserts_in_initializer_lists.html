prefer_asserts_in_initializer_lists

Group: style

Maturity: stable

  • List of Lint Rules

WARNING Putting asserts in initializer lists is only possible using an experimental language feature that might be removed.

DO put asserts in initializer list for constructors with only asserts in their body.

GOOD:

class A {
  A(int a) : assert(a != null);
}

BAD:

class A {
  A(int a) {
    assert(a != null);
  }
}
