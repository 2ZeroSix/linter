recursive_getters

Group: style

Maturity: stable

View the Project on GitHub dart-lang/linter

  • See the Style Guide
  • List of Lint Rules

DON'T Return the property itself in a getter body, this can be due to a typo.

BAD:

  int get field => field; // LINT

BAD:

  int get otherField {
    return otherField; // LINT
  }

GOOD:

  int get field => _field;
